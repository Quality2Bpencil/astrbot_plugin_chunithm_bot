# Ku2uka Bot 使用指南

## 账号绑定

目前只支持落雪 (https://maimai.lxns.net)。

一些功能无需手动绑定账号，只需要在 落雪官网 $\rightarrow$ 账号详情 $\rightarrow$ 第三方应用 $\rightarrow$ 第三方账号绑定 中绑定 $\text{QQ}$ 号即可。

一些功能需要手动绑定账号。与 $\text{bot}$ 私聊发送“$\text{/bind}$”即可。

## 指令

<font color=red>红色</font>的指令为需要手动绑定账号的指令。

| 指令 | 说明 |
| ---- | ---- |
| $\text{/help}$ | 查看使用指南 |
| $\text{/bind}$ | 绑定账号 |
| $\text{xxx}$ 是什么歌 | 查询歌曲 |
| $\text{/b30}$ 或 $\text{/b50}$ | 查询 $\text{B30 + N20}$ |
| $\text{/max30}$ 或 $\text{/max50}$ | 查询理论 $\text{Rating}$ 表 |
| $\text{/dsb}$ [难度或定数] | 查询指定难度或定数的定数表 |
| <font color=red>$\text{/list}$ [难度或定数]</font> | 查询指定难度或定数的谱面的所有分数 |
| <font color=red>$\text{/overpower level/version/genre}$</font> | 查询全曲 $\text{OVERPOWER}$ （只计算最高难度谱） <br> 参数为分类（等级，版本，曲目类型），默认为等级 |
